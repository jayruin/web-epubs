Running

Global Command Line Arguments

--projects-directory
Defaults to projects
--expanded_epubs_directory
Defaults to docs
--packaged_epubs_directory
Defaults to dist
--epubcheck-directory
Defaults to epubcheck
--logs-directory
Defaults to log
--bundles-directory
Defaults to bundles
positionial argument projects
List of strings of project names (defaults to empty list). Each project should be a subdirectory of --projects-directory.
--all
Boolean flag (which defaults to false if absent). If present, projects will be ignored and all possible projects will be processed within the context of the running module.

epub.build

python -m epub.build.{TYPE}

Create an expanded ePub of type {TYPE}. Types available by default are epub3 and epub2.

Additional command line arguments:

-b, --bundles
List of strings of the bundles to include (defaults to empty list). Each bundle should a subdirectory of --bundles-directory. Bundles contain files to include in the output ePub, such as shared CSS and JS files.

epub.pack

python -m epub.pack

Packages ePubs in --expanded_epubs_directory and places the packaged ePubs in --packaged_epubs_directory.

Additional command line arguments:

-t, --type
Type (see epub.build) of the ePub to pack (defaults to epub3).
-c, --compression
Integer in range [0, 9] specifiying the compression level to use (defaults to 0, no compression).

epub.check.install

python -m epub.check.install

Install EPUBCheck to the --epubcheck-directory.

epub.check

python -m epub.check

Validate the project ePubs using EPUBCheck and write the results to --logs-directory.

Additional command line arguments:

-t, --type
Type (see epub.build) of the ePub to check (defaults to epub3).
