Building ePubs

Manually Building

Install or update EPUBCheck

python -m epub.check

Create ePub contents from template

The .xhtml files can be deployed on the web (Github Pages) without any modification.

usage: python -m templates.simple [-h] projects [projects ...]

Create ePub contents from template.

positional arguments:
    projects    List of projects. Each project should be a subdirectory of the html directory.

optional arguments:
    -h, --help  show this help message and exit

Zip ePub contents into a single .epub file

usage: python -m epub.zipping [-h] projects [projects ...]

Zip ePub contents into a single .epub file.

positional arguments:
    projects    List of projects. Each project should be a subdirectory of the html directory.

optional arguments:
    -h, --help  show this help message and exit

Automated Building

There is also a GitHub Workflow Create Release ePubs found in create-release-epubs.yml which will automatically build ePubs when unzipped files are uploaded to epub/.
